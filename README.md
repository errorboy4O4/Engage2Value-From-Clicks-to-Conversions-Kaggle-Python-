# Engage2Value-From-Clicks-to-Conversions-Kaggle-Python-

Analyzed session-level digital commerce data to predict customer purchase value based on user behavior, traffic source, device attributes, and geographic features.

Performed data preprocessing, missing value handling, and consistent categorical encoding, and trained regression models including Ridge Regression, LightGBM, and XGBoost.

Evaluated models using RMSE on a validation set and applied model blending to improve purchase value prediction performance.
